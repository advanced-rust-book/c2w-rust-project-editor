Use it on your page

Create one shared c2w terminal bridge, then mount as many project editors as you need. Each editor owns its project folder while sharing the same browser container and Rust runtime.

const runtime = new RustContainerWrapper();
const terminal = new C2WRustEditor.SharedTerminalBridge({
  terminalElement: "#terminal-amd64-debian",
});

C2WRustEditor.createProjectEditor({
  root: "#my-editor",
  runtime,
  terminal,
  project: {
    id: "my-editor",
    title: "My Cargo project",
    projectDir: "/root/my-cargo-project",
    tabs: [
      { id: "cargo", label: "Cargo.toml", path: "Cargo.toml", defaultContent: cargoToml },
      { id: "main", label: "src/main.rs", path: "src/main.rs", defaultContent: mainRs },
    ],
  },
});
